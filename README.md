# Smart Study Scheduler

Smart Study Scheduler is a full-stack web application designed to help students
plan, track, and manage their study schedules efficiently.  
The project focuses on productivity, progress tracking, and future AI-driven
schedule optimization.

---

## Features

- Create daily and subject-wise study schedules
- Mark tasks as completed with real-time progress tracking
- Persistent schedules using backend storage
- Clean and responsive frontend UI
- Scalable backend architecture
- Designed for future AI/NLP-based recommendations
